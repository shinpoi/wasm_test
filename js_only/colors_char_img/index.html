着色字符画

说明：图 -> 沙雕着色的字符画

  1. 上传要转换的图 (source image)
  2. 填写参数（保持默认也行...）
  3. 点 "gen image" 生成
  4. 点 "download result" 下载结果

source image:

width (px): - 生成图宽(像素)

height (px): - 生成图高(像素)

font_size (px): - 字符尺寸(像素) 注意：生成图的实际尺寸会被裁剪为字符尺寸的整数倍

strings: - 生成图用字符串

back color: - 背景色, 用12或者24位色标，或者十进制rgba(r,g,b,a)表示。没做合法检查，乱输出错了俺不管...

  • 色标示例:
  • 12位色标: #rgb - #000(纯黑), #fff(纯白), #f00(纯红)
  • 24位色标: #rrggbb - #000000(纯黑), #ffffff(纯白), #00ff00(纯绿)
